Get Started

3ステップで、すぐに

セットアップは数分。あとは好きなデバイスから接続するだけ。

1
Gateway を起動
macOS / Linux でワンコマンド実行。トークン生成からサービス登録まで全自動。
$ npx zenterm-gateway
2
QR コードをスキャン
表示される QR をアプリでスキャン。接続情報が自動入力。
3
接続完了
iPhone やブラウザーからターミナルを操作。どこからでも、いつでも。
