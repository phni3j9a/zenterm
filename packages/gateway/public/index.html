Features

主な機能

— 必要なものが、すべて揃っている
PC と同じターミナル
フル機能のターミナルをスマホで。日本語入力も、5,000 行の履歴も、そのまま使える。
切れても続く作業
電波が途切れても、アプリを閉じても、作業はサーバーで動き続ける。再接続すれば元の画面がそのまま。
ファイルも直接操作
コードの確認、設定ファイルの編集、ファイルのアップロード。ターミナルを開かなくても操作できる。
サーバーの状態が見える
CPU、メモリ、ディスク、温度。サーバーの健康状態をリアルタイムで確認。
QR コードで即接続
QR コードをスキャンするだけ。IP アドレスやポート番号を手で入力する必要はない。
1 行で準備完了
ターミナルで一行コマンドを実行するだけ。サーバー再起動後も自動で立ち上がる。

FAQ

よくある質問

Q. 月額料金はかかりますか?
いいえ。アプリも Gateway も無料で、追加課金はありません。あなたのサーバー上で動くので、運用コストはサーバー代だけ。Raspberry Pi のような低コスト機でも動きます。
Q. Android で使えますか?
現在、専用アプリは iOS / iPadOS に対応しています。Android では PC ブラウザ向けの Web UI から接続できます。ネイティブの Android アプリは検討中です。
Q. AI 機能は内蔵されていますか?
ZenTerm 自体はターミナルなので AI は搭載していません。サーバーに claude や codex などの Coding Agent をインストールしておき、それを ZenTerm 越しに操作する設計です。AI の選択はあなた次第。
Q. セキュリティはどうなっていますか?
通信は WebSocket over TLS、認証はサーバーが発行する Bearer Token のみ。ZenTerm 側にアカウント DB はなく、認証情報はあなたのサーバーとアプリの間だけに閉じます。
Q. 外出先から自宅サーバーに繋ぐには?
Tailscale などの VPN や、Cloudflare Tunnel・DDNS+ポート開放のいずれかで、サーバーに外部からアクセスできるようにします。VPN 経由がおすすめ。
Q. 必要なサーバースペックは?
Node.js が動く macOS または Linux であれば OK。Gateway 自体は軽量で、Raspberry Pi クラスでも動作します。AI を走らせる場合は、AI が必要とするスペックが基準になります。
