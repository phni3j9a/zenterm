## Phase 1 (完了)

- [x] packages/web 初期セットアップ (React + Vite)
- [x] Zen テーマトークン (CSS Custom Properties, dark/light)
- [x] レイアウト (Header, Sidebar, StatusBar)
- [x] 認証 (Bearer token ログイン)
- [x] セッション管理 (一覧, 作成, リネーム, 削除)
