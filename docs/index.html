Features

主な機能

— 必要なものが、すべて揃っている
Full Terminal
本格ターミナルをiPhoneに。Vim、top、htop — 何でも動く。
tmux Sessions
接続が切れても作業は消えない。後から別のデバイスで続きを再開。
File Browser
サーバーのファイルをスマホで直感操作。編集もダウンロードもワンタップ。
System Monitor
CPU、メモリ、温度。サーバーの健康状態を一目で確認。
QR Pairing
面倒な設定はゼロ。QRコードをかざすだけで接続完了。
One-command Setup
ターミナルに一行コピーするだけ。あとは全自動。
Get Started

3ステップで始める

たった3ステップ、数分で完了。あとは好きなデバイスからサーバーを操るだけ。

1
Gateway を起動
macOS または Linux のサーバーで実行。トークン生成と初期設定が自動で完了する。
$ npx -y zenterm-gateway@latest
2
QRコードをスキャン
起動時にターミナルへ表示されるQRコードをアプリでスキャン。接続情報が自動入力される。
3
接続完了
iPhone、Android、またはブラウザからサーバーのターミナルを操作。どこからでも作業できる。
