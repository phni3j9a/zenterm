Features

主な機能

— 必要なものが、すべて揃っている
PCと同じターミナル
フル機能のターミナルをスマホで。日本語入力も、5000行の履歴も、そのまま使える。
切れても続く作業
電波が途切れても、アプリを閉じても、作業はサーバーで動き続ける。再接続すれば元の画面がそのまま。
ファイルも直接操作
コードの確認、設定ファイルの編集、ファイルのアップロード。ターミナルを開かなくても操作できる。
サーバーの状態が見える
CPU、メモリ、ディスク、温度。サーバーの健康状態をリアルタイムで確認。
QRコードで即接続
QRコードをスキャンするだけ。IPアドレスやポート番号を手で入力する必要はない。
1行で準備完了
ターミナルで一行コマンドを実行するだけ。サーバー再起動後も自動で立ち上がる。
画像添付
カメラやフォトライブラリから画像を添付。スクリーンショットやエラー画面をそのまま共有できる。
Get Started

3ステップで始める

複雑な設定は不要。数分でiPhoneからサーバーを操作できるようになる。

1
Gateway を起動
macOS または Linux のサーバーで実行。トークン生成と初期設定が自動で完了する。
$ npx -y zenterm-gateway@latest
2
QRコードをスキャン
起動時にターミナルへ表示されるQRコードをアプリでスキャン。接続情報が自動入力される。
3
接続完了
iPhone、Android、またはブラウザからサーバーのターミナルを操作。どこからでも作業できる。
