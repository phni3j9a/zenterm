Borderless Philosophy

無 -- 線を消す。カードもヘアラインも廃止。セクションラベルの上に広い余白を取ることで、線なしでもグルーピングが伝わる。近接の原則だけで構造化する。


列 -- 行の力。セッション・ファイル・設定をすべて同じ行コンポーネントで表現。dot + テキスト + 右端アクセサリの統一レイアウト。行にhover時だけ背景色が浮かぶ。


区 -- 余白でセクションを区切る。Active / Recent のラベル上に大きなマージンを設け、視覚的な「段落」を作る。線に頼らない上質な構造化。


数 -- System Status は数字だけ。バーチャートやカード枠を省き、22% / 37% / 14% の数字を縦区切りで並べる。一瞥で読める。


余 -- 引き算の極致。カード・線・影をすべて取り去った先に残るのは、テキストと余白だけ。「何もない」が最も禅的なUI。
